Porównanie programowania strukturalnego i obiektowego — notatka

Wersja z kolorowaniem składni (Prism.js). Przykłady kodu w C#, HTML i CSS są podświetlone.

Przykład: abstrakcja (C#)

Klasa abstrakcyjna Wielokat
abstract class Wielokat
{
    protected int[] boki;

    public int IleBokow()
    {
        return boki.Length;
    }

    public int Obwod()
    {
        int obwod = 0;
        foreach (int bok in boki)
        {
            obwod += bok;
        }
        return obwod;
    }

    public abstract int Pole();
}

Przykład: dziedziczenie (C#)

Klasa Kwadrat dziedzicząca po Wielokat
class Kwadrat : Wielokat
{
    public Kwadrat(int a)
    {
        boki = new int[4];
        boki[0] = boki[1] = boki[2] = boki[3] = a;
    }

    public override int Pole()
    {
        return boki[0] * boki[0];
    }
}
