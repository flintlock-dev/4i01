Zadanie T21a z wykorzystaniem Break

Autor: Michał Knurowski

Napisz program w JavaScript, który: Wczyta od użytkownika dodatnią liczbę całkowitą n za pomocą funkcji prompt. Sprawdzi, czy wprowadzona wartość jest poprawna (dodatnia liczba całkowita). Używając pętli for, wyświetli kwadraty liczb od 1 do n za pomocą document.write, każde w nowej linii. Przykład działania programu: Jeśli użytkownik wpisze 3, program wyświetli: Kwadrat liczby 1 to: 1 Kwadrat liczby 2 to: 4 Kwadrat liczby 3 to: 9 Porady dla uczniów: Wczytywanie danych: Użyj prompt("Wiadomość"), aby wczytać dane od użytkownika. Wynik jest tekstem (stringiem). Skorzystaj z parseInt(), aby zamienić tekst na liczbę całkowitą. Sprawdź, czy liczba jest dodatnia i czy nie jest NaN za pomocą warunku if (isNaN(n) || n <= 0). Pętla for: Ustaw zmienną sterującą (np. i) od 1 i iteruj do n włącznie: for (let i = 1; i <= n; i++). W każdej iteracji oblicz kwadrat liczby i (np. i * i). Wyświetlanie wyników: Użyj document.write() do wyświetlenia wyniku. Aby przejść do nowej linii, dodaj
na końcu tekstu, np. document.write("Wynik: " + wynik + "
"). Upewnij się, że tekst jest czytelny, np. "Kwadrat liczby X to: Y". Testowanie: Sprawdź program dla różnych danych wejściowych, np. 3, 0, -1, abc. Jeśli dane są niepoprawne, wyświetl komunikat, np. "Proszę podać dodatnią liczbę całkowitą.".
